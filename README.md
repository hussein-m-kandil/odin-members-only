# Odin Members Only

An exclusive clubhouse developed as part of the [**Node.js course at The Odin Project**](https://www.theodinproject.com/paths/full-stack-javascript/courses/nodejs). Only the members can see who the author of a post is, but outside they can only see the story and wonder who wrote it.
